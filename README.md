# Spring Boot MongoDB BackEnd 


Spring Boot project with MongoDB backend for CRUD operations over users with JWT authentication.

## Technology Stack:

- Spring Boot
- MongoDB
- JWT Authentication

 ### Build:
 ```bash 
 ./mvnw clean package
```

### To run docker:
```bash 
cd docker &&  docker-compose up -d
```

### To stop docker:
```bash 
cd docker && docker-compose down -v
```

## Requirements:
 - Maven 3.8.0
 - Java 8
 - Docker and Docker compose

### To run locally:
```bash 
./mvnw spring-boot:run
```
